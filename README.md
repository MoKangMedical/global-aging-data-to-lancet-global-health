# 全球老龄化数据到柳叶刀全球健康

AI驱动的全球老龄化数据分析 — 人口健康到顶级期刊

## 项目目标

将全球老龄化数据（WHO/GBD/UN Population）转化为柳叶刀全球健康期刊发表的研究论文。

## 核心能力

- 全球老龄化数据自动获取
- 健康期望寿命分析
- 疾病负担趋势分析
- 跨国比较研究

## 快速开始

    git clone https://github.com/MoKangMedical/global-aging-data-to-lancet-global-health.git
    cd global-aging-data-to-lancet-global-health
    pip install -r requirements.txt
    python src/main.py --region "asia" --metric "DALYs"
